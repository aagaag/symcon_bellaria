# symcon_bellaria

Project workspace for `symcon_bellaria`.

## Structure

- `src/` - source code and application modules
- `tests/` - tests and validation scripts
- `docs/` - project notes and design documentation
- `.githooks/` - local Git hooks for project sync automation
- `scripts/` - setup and maintenance scripts

## Getting Started

Add the project stack, runtime, and setup instructions here as the implementation takes shape.

## GitHub Sync

This project is intended to sync with `aagaag/symcon_bellaria`.

Once Git is installed and the GitHub repository exists, run:

```powershell
.\scripts\install-auto-sync.ps1
```

After that, each local commit will automatically push to GitHub.
